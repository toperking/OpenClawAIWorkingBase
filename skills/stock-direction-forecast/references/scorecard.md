# 漲跌預判計分卡（短線）

總分範圍：-10 到 +10

## 指標與權重
1. 趨勢（-3~+3）
- 收盤站上短均/長均且均線上彎：+2~+3
- 跌破關鍵均線且走弱：-2~-3

2. 量能（-2~+2）
- 上漲帶量、回檔縮量：+1~+2
- 下跌帶量、反彈無量：-1~-2

3. 籌碼/資金流（-2~+2）
- 外資/法人連續買超：+1~+2
- 連續賣超：-1~-2

4. 事件催化（-2~+2）
- 財報/指引/產業消息正向：+1~+2
- 政策/需求/地緣風險負向：-1~-2

5. 波動與風險（-1~+1）
- 波動收斂且結構穩定：+1
- 波動擴大且不確定上升：-1

## 分數映射
- +4 以上：偏多（上漲機率約 58%~70%）
- -3 ~ +3：震盪（雙向機率接近）
- -4 以下：偏空（下跌機率約 58%~70%）

## 信心標籤
- 高：資料完整 + 指標同向 + 事件清晰
- 中：資料大致完整，但有1-2項矛盾
- 低：資料缺漏或重大事件前（如財報前夕）
